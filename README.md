# Spaceborne lidar for vegetation assessment

[![DOI](https://zenodo.org/badge/DOI/10.5281/zenodo.19099501.svg)](https://doi.org/10.5281/zenodo.19099501)

This repository contains instructional materials and code for the workshop **“Spaceborne lidar for vegetation assessment.”**  
The purpose of this project is to introduce students and researchers to the use of **spaceborne lidar data**—specifically ICESat‑2 and GEDI—for vegetation and forest assessment.  

The notebook walks through:
1. Accessing and interpreting ICESat‑2 data products  
2. Using Google Earth Engine for geospatial analysis  
3. Running workflows in Google Colab for open‑source data science  
4. Applying machine learning for model generation and evaluation

## 📝 Project Abstract

The goal of this workshop is to introduce students to **spaceborne lidar data for vegetation assessment**.  
Participants will learn to understand and apply tools such as:

1. **ICESat‑2 data products**  
2. **Google Earth Engine** for cloud‑based geospatial processing  
3. **Google Colab** for open‑source data science workflows  
4. **Machine learning** algorithms for model development and evaluation  

These materials support open, reproducible research in forest ecology, lidar analysis, and remote sensing.
